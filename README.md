# AllpaOps — Inteligencia Operativa desde la Raíz 🌿🚀

**AllpaOps** es una plataforma SaaS Agritech diseñada para digitalizar el "último kilómetro" operativo del agroexportador peruano. Nuestra misión es erradicar la **trampa de la manualidad**, transformando procesos ciegos de campo en activos estratégicos auditables y en tiempo real.

---

## ⚠️ The Problem: El Techo de Cristal Agroindustrial

Perú es una potencia agroexportadora global, sin embargo, el corazón de su operación sigue atrapado en el siglo pasado:
- **Dependencia de Manualidad:** Coordinación crítica de campo vía papel, WhatsApp y hojas de cálculo fragmentadas.
- **Latencia Crítica:** Datos que tardan de 24 a 48 horas en llegar a la oficina central, impidiendo la toma de decisiones ágil.
- **Riesgo Regulatorio:** Trazabilidad deficiente ante estándares globales exigentes como **GlobalG.A.P.** y **EUDR 2025**.
- **Ceguera de Costos:** Hasta un 28% de horas fantasma no auditadas debido a sistemas de tareo manual.

---

## ✨ The Solution: Inteligencia en la Raíz

AllpaOps actúa como la **capa operativa inteligente** que conecta el surco con la decisión ejecutiva:
- **Offline-First Architecture:** Captura de datos garantizada incluso en zonas de nula conectividad, con sincronización inteligente.
- **Audit-Ready Ecosystem:** Evidencia digital nativa para cumplimiento de normativas internacionales de exportación.
- **Interoperabilidad Enterprise:** APIs diseñadas para hablar el lenguaje de los ERPs corporativos (SAP, Oracle, ERPs nacionales).
- **Operational Clarity:** Visualización espacial (GIS) y métricas de rendimiento por lote en tiempo real.

---

## 🚀 Key Features

- **Digital Tareo:** Control exacto de asistencia, rendimientos y actividades de personal en campo.
- **Smart Offline Sync:** Algoritmo de sincronización que preserva la integridad de los datos sin depender de señal constante.
- **GIS Field Management:** Mapeo y georreferenciación de lotes para una trazabilidad parcelaria precisa.
- **Compliance Engine:** Generación automática de reportes de trazabilidad listos para auditorías internacionales.
- **Real-Time Intelligence:** Dashboards ejecutivos con latencia cero entre el campo y la gerencia.

---

## 🛠️ Tech Stack

El proyecto utiliza una arquitectura modular moderna enfocada en el rendimiento y la experiencia de usuario cinematográfica:

- **Frontend Core:** React 18 + Vite
- **Animaciones:** GSAP (GreenSock Animation Platform) para transiciones fluidas tipo Keynote.
- **Visualización:** Apache ECharts para analítica avanzada de impacto.
- **Estilos:** Tailwind CSS + CSS Variables para un sistema de diseño consistente.
- **Tipografía:** Syne (Headers) & DM Sans (Body) para una estética técnica/premium.
